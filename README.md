# Sistema de Biblioteca Digital

Proyecto desarrollado en Python que simula un sistema de gestión de biblioteca utilizando Programación Orientada a Objetos.

## Funcionalidades

- Añadir libro
- Quitar libro
- Registrar usuario
- Prestar libro
- Buscar libro por título
- Buscar libro por autor
- Buscar libro por categoría
- Listar libros de un usuario

## Ejecutar el programa

python principal.py
